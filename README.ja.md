# フリーBGMプレイヤー

フリー音源サイト「BGMer」のBGMを再生できるWebアプリケーションです。ジャンル、キーワード、カテゴリーによる検索、ショート/ロング音源の切り替え、ループ再生などの機能を提供しています。

## デモ
https://code4fukui.github.io/free-bgm-player/

## 機能
- ジャンル、キーワード、カテゴリーによる検索
- ショート/ロング音源の切り替え
- ループ再生
- 音源ページへのリンク

## 必要環境
特に必要ありません。Web上で動作します。

## 使い方
ウェブブラウザからデモサイトにアクセスし、検索条件を指定して楽曲を選択して再生することができます。

## データ・API
本アプリケーションでは、「BGMer」のサイトから楽曲情報を取得しています。
