# free-bgm-player

フリーBGMサイト「[BGMer](https://bgmer.net/)」の音楽を再生できる、シンプルなシングルファイルHTMLプレイヤーです。

## デモ

**https://github.com/code4fukui/free-bgm-player

## 機能

- ジャンル、キーワード、カテゴリによる音楽のフィルタリング
- タイトル、説明、タグを対象としたキーワード検索
- 楽曲のSHORT版とLONG版の切り替え
- ループ再生による連続再生
- 各楽曲のBGMerオリジナルページへのリンク

## 使い方

インストールは不要です。Webブラウザで[デモページ](https://code4fukui.github.io/free-bgm-player/)を開くだけで利用できます。

1. ドロップダウンメニューと検索ボックスを使って、好みのBGMを探します。
2. `ショート` チェックボックスをオンにすると、楽曲のショートバージョンが再生されます。
3. `ループ` チェックボックスをオンにすると、ループ再生が有効になります。
4. テーブルの任意の行をクリックすると、再生が開始されます。

## データソース

このプレイヤーは、フリーBGMサイト「[BGMer](https://bgmer.net/)」からスクレイピングした音楽データを使用しています。楽曲情報は`music_list.csv`にまとめられています。デモ環境では、音声ファイルは別のサーバーでホストされています。

音楽のライセンス情報については、[BGMerの利用規約](https://bgmer.net/terms)をご参照ください。

## データの更新

同梱されているDenoスクリプトを使用して、楽曲リストと音声ファイルを更新できます。

**前提条件:**
- [Deno](https://deno.land/)がインストールされていること。

**手順:**
1. **楽曲リストの更新:** このスクリプトはBGMerから最新の音楽メタデータをスクレイピングし、`music_list.csv`を作成します。
    ```sh
    deno run -A download.js
    ```
2. **音声ファイルのダウンロード:** このスクリプトは`music_list.csv`を読み込み、対応する音声ファイルを`data/short/`および`data/long/`ディレクトリ（Gitの管理対象外）にダウンロードします。
    ```sh
    deno run -A downloadMusic.js
    ```
